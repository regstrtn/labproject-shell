# A simple unix shell implemented in C
Project to create a basic unix shell. 
Compile and execute the shell.c file to start the main shell. 
Implements the following functionality: 
  1. ls
  2. mv
  3. cd
  4. rm
  5. mkdir
  6. tail
  7. ps
  8. pwd

To exit, type "exit".
